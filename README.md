## Running Locally

1. ### Clone the repository

```bash
git clone https://github.com/CornerMan-Hyacinth/erprex.git
```

2. ### Navigate into the project

```bash
cd erprex
```

3. ### Install dependencies

```bash
npm install
```

4. ### Start the development server

```bash
npm run dev
```

5. ### Visit:
   [http://localhost:3000](http://localhost:3000)
